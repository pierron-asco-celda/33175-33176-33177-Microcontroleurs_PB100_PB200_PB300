Utilisation Firmata


1 - Installer le firmware Firmata sur l'ARDUINO

IDE Arduino
  Sketch/Include Librairie/Manage Librairie
    Chercher Firmata
      installer la dernière version de Firmata built-in by Firmata developers
  Ouvrir Examples/Firmata/StandardFirmata
    compiler et téléverser sur l'arduino nano (bien selectionner le type de carte et le port)

2 - Installer python et les dépendances suivantes
    pyFirmata   :   https://pypi.org/project/pyFirmata/
